# TRMNL Terminus

Terminus is a self-hosted TRMNL BYOS server. This app packages Terminus `0.60.0`, PostgreSQL, and Valkey in one Home Assistant app container and stores persistent state under Home Assistant app storage.

Open the Web UI after starting the app, then configure your TRMNL devices to use the same URL as the app `api_uri` option.

For setup details, see the [full app documentation](DOCS.md).
